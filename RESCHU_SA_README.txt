RESCHU-SA Eclipse Java Neon configuration

1. Please install Eclipse Java Neon (http://www.eclipse.org/downloads/eclipse-packages/)

2. Please download all RESCHU-SA related files

3. Start Eclipse Java Neon

4. Go to File > Import > Projects from Folder or Archive > select RESCHU_Eclipse to import

5. Go to Project > Properties > Java Build Path > make sure all paths in "libraries" and "Order and Export" are correctly configured

6. Go to Run Configurations > Java Application > Main > set Main class to "reschu.app.AppMain"

7. (recommended) Go to Run Configurations > Java Application > Arguments > set VM arguments to "-Xmx2048m"

8. If you have other questions, please contact haibei.zhu@duke.edu
